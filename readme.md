# Git學習筆記

### Git基本安裝及設定 

1. 確認git安裝及版本
```bash=
$ git --version
```
[git](https://git-scm.com/)
[Sourcetree](https://www.sourcetreeapp.com/)

2. 設定環境變數

```bash=
#設定使用者名稱
$ git config --global user.name "使用者名稱"

#設定使用者email
$ git config --global user.email "使用者email"

#確認設定的情況
$ git config --list

#git的設定檔
$ cat ~/.gitconfig
```
